# This is a script to scrape Amazon items into a CSV file

## Usage:  `python3 main.py <search+item>`

- It is recommended to use a Python virtual environment

### Required Packages:

```
pip install seleniumbase
pip install beautifulsoup4
pip install lxml
pip install tqdm
pip install pandas
```
